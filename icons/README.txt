APP ICONS NEEDED FOR PWA

To generate the app icons:
1. Open generate-icons.html in your web browser
2. Click "Download 192x192" to download the first icon
3. Click "Download 512x512" to download the second icon  
4. Save both files in this directory as:
   - icon-192.png
   - icon-512.png

Alternatively, you can create custom icons:
- 192x192 pixels PNG
- 512x512 pixels PNG
- Use the brand color #e0724c (orange) as background
- White "K" letter in the center

The SVG template is available in icon.svg if you want to customize it.
